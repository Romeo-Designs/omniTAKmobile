## How to Use

### Change Your Unit Type

1. Open **Settings** (gear icon)
2. Scroll to **Position Broadcasting** section
3. Tap **Select Unit Type**
4. Choose your affiliation (Friendly, Hostile, Neutral, Unknown)
5. Choose your dimension (Ground, Air, Sea, Space, Subsurface)
6. Choose your unit type from the organized list
7. Tap **Save**

Your position will immediately broadcast with the new unit type.

### Quick Access Methods

#### Method 1: Recent Selections

1. Open unit type selector
2. Look at the "RECENT" section at top
3. Tap your recently used unit type

#### Method 2: Favorites

1. Open unit type selector
2. Tap the **star icon** in the toolbar (top-left)
3. View only your favorited unit types
4. Tap to select

#### Method 3: Search

1. Open unit type selector
2. Type in the search bar
3. See filtered results instantly
4. Tap to select

### Managing Favorites

**Add to Favorites:**

- Tap the star icon next to any unit type
- Star turns yellow when favorited

**Remove from Favorites:**

- Tap the filled yellow star icon
- Star becomes empty outline

**View All Favorites:**

- Tap the star button in the toolbar
- Toggle back to see all unit types

## Understanding Unit Type Codes

Your unit type is represented by a CoT (Cursor on Target) code like:

```
a-f-G-U-C-I
```

Breakdown:

- `a` = Atom (position/status)
- `f` = Friendly
- `G` = Ground
- `U` = Unit
- `C` = Combat
- `I` = Infantry

The app handles this complexity automatically - just choose your unit type from the visual picker!

## Troubleshooting

**Q: My unit type isn't showing on the map**
A: Check your position broadcasting is enabled in Settings

**Q: Other users see wrong icon**
A: Ensure you pressed "Save" after selecting unit type

**Q: Can't find a specific unit type**
A: Use search - type part of the name or check the category sections

**Q: My favorites disappeared**
A: Check if you're logged into the same device/iCloud account

**Q: Recent selections not showing**
A: Recent list only shows after you've used multiple unit types

## Integration with ATAK/WinTAK

Your unit type selection is fully compatible with:

- ATAK (Android Team Awareness Kit)
- WinTAK (Windows TAK)
- iTAK (iOS TAK)
- TAK Server

Other TAK clients will see your unit type correctly rendered using standard MIL-STD-2525 symbology.

## Advanced: Custom Unit Types

Currently, the app supports all standard MIL-STD-2525B unit types. Custom unit types and advanced symbology features are planned for future releases.
